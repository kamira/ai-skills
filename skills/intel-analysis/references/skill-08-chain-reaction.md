# SKILL-08|連鎖反應推演

## 概述
推演行動→反應→再行動的動態鏈,評估升級與收斂路徑,用跨域傳導矩陣管理多維度同步傳導與反饋迴路。

## 對應框架
**步驟七 + 系統層 B**|十步驟分析框架

## 核心問題
> 動作之後局面如何改變?行動→反應→再行動的動態鏈是什麼?

## 核心原則
> 分析不是靜態快照。A 做了 X 之後,整個局面就變了——B 會反應,C 會調整,連 A 自己的下一步選項也會改變。
> **不考慮反饋迴路的分析,等於在真空中下棋。**

## 執行步驟

**1. 行動→反應鏈**
- 如果 A 打出某張牌,B 的選項如何改變?
- B 的反應如何進一步影響 C?

**2. 升級與收斂路徑**
- 什麼條件下事態螺旋升級?
- 什麼條件下事態趨於穩定或降溫?

**3. 跨領域傳導**
- 軍事行動如何外溢到經濟領域?
- 經濟制裁如何影響政治決策空間?
- 資訊操作如何改變軍事行動的政治授權?

**4. 反饋修正**
- 連鎖反應如何改變決策者的認知框架與風險偏好?
- 連鎖反應如何重塑硬約束(讓原本不可用的手段變可用,或反之)?

## 系統層 B:跨域傳導矩陣

當事件同時觸發多個領域時,使用傳導矩陣管理多維度同步傳導。

**矩陣結構**:橫軸是行為者,縱軸是傳導領域(軍事、經濟、政治、資訊)

**操作步驟**:
1. 先填對角線:每個行為者在自己主導領域的直接行動
2. 再填跨格:該行動如何傳導到其他領域
3. 標記回饋迴路:哪些跨格傳導會反過來影響原始行動者

```
傳導矩陣(範例格式)

           │ 軍事   │ 經濟   │ 政治   │ 資訊
───────────┼────────┼────────┼────────┼────────
行為者 A   │ [直接] │ [傳導] │ [傳導] │ [傳導]
行為者 B   │ [傳導] │ [直接] │ [傳導] │ [傳導]
行為者 C   │ [傳導] │ [傳導] │ [直接] │ [傳導]
```

## 推演模板

```
初始行動:[行為者 A 採取手段 X]

【第一輪反應】
行為者 B:[反應描述] → 改變 B 的可用選項:[]
行為者 C:[反應描述] → 改變 C 的可用選項:[]

【升級路徑】
觸發條件:[]
升級至:[下一個烈度等級]

【收斂路徑】
觸發條件:[]
收斂至:[穩定狀態描述]

【跨域傳導】
軍事 → 經濟:[]
經濟 → 政治:[]
政治 → 軍事授權:[]

【反饋修正】
哪些硬約束被重塑:[]
哪些認知偏好被強化或削弱:[]
```

## 輸出格式

```
初始行動:[行為者 A 採取手段 X]
反應鏈:[B 反應] → [C 反應]
升級路徑:觸發條件 [] → 升級至 []
收斂路徑:觸發條件 [] → 收斂至 []
跨域傳導:[主要傳導路徑]
反饋修正:[被重塑的約束 / 被強化的偏好]
```

## 對手代言人機制(Adversary Advocate)

在連鎖反應推演中,必須至少對一個關鍵行為者執行「對手代言人」切換,避免以自己的邏輯替對手思考。

### 核心原則
> **不是「如果我是對手,我會怎麼做」,而是「如果我擁有對手的資訊、利益、約束和認知偏誤,我會怎麼做」。**
> 前者仍然是鏡像思維。後者才是真正的對手模擬。

### 操作步驟

**1. 選擇代言對象**
- 選擇在本次推演中**最可能被誤判的行為者**(通常是對手方或利益與分析者最不同的一方)
- 每次推演至少選一個行為者執行代言

**2. 切換資訊環境**
回答以下問題,從對手的角度填入:
```
我(作為 [行為者])此刻知道什麼:
- 已確認的事實(從對手的情報管道能看到什麼):[]
- 我不知道的(對手可能缺乏的資訊):[]
- 我可能誤判的(對手可能持有的錯誤認知):[]
```

**3. 切換利益與約束框架**
```
我(作為 [行為者])的核心利益排序:
1. [來自 SKILL-04 行為者卡片,但從對手視角重新排序]
2. []
3. []

我面臨的約束(從對手內部看):
- 國內政治:[]
- 資源與能力:[]
- 組織慣性:[]
- 時間壓力:[]
```

**4. 切換認知框架**
```
我(作為 [行為者])最可能有的認知偏誤:
- 歷史類比:我最可能用什麼過去經驗來理解當前局勢?[]
- 確認偏誤:我最傾向相信什麼?[]
- 對敵方的看法:我認為敵方最可能做什麼?為什麼?[]
```

**5. 以對手身份做出決策**
```
基於以上資訊、利益、約束和認知框架,
我(作為 [行為者])最可能的下一步行動是:[]

原因:[]

這與「從分析者角度預測的對手行為」有何不同:[]
差異的來源是:[資訊差/利益差/認知差]
```

### 代言人輸出格式

```
【對手代言人分析】
代言行為者:[]
資訊環境差異:[對手知道什麼、不知道什麼、可能誤判什麼]
利益排序(對手視角):[]
認知框架:[歷史類比/確認偏誤/對敵看法]
對手最可能的行動:[]
與分析者預測的差異:[]
差異來源:[]
對推演的影響:[是否改變連鎖反應的路徑或機率]
```

### 何時必做
- 推演中出現「對手不可能這樣做」的判斷時 → **必做**,因為這正是鏡像思維最危險的時刻
- 對手行為出現「意外」時(SKILL-06 一致性檢查不通過)→ **必做**
- 對手面臨的國內政治或派系壓力不透明時 → **建議做**

### 何時可省略
- 對手行為與利益邏輯高度一致、且有充分可觀測證據支撐時

## 注意事項
- ⚠️ 若某個跨格傳導路徑特別強,應在 SKILL-09 和 SKILL-10 中重點呈現
- ✅ 完成後接 **SKILL-09**(情境預測)
- 💡 反饋迴路可能讓 SKILL-05 的牌桌發生變化,必要時回頭更新
- 💡 連鎖推演中至少對一個關鍵行為者執行**對手代言人機制**,避免鏡像思維
