# SKILL-18|長期結構趨勢分析

## 概述
識別與評估超越單一事件的長期結構性趨勢——人口結構、能源轉型、軍工產能、聯盟漂移等——作為所有事件分析的背景基準與約束條件。

## 對應框架
**擴展模組**|為所有 SKILL 提供長期結構性背景,尤其與 SKILL-04(行為者分析)、SKILL-05(約束篩選)、SKILL-09(情境預測)互動

## 核心問題
> 哪些長期結構性趨勢正在改變局勢的底層條件?這些趨勢如何約束或擴大行為者的選項?

## 核心原則
> **事件是水面上的波浪,結構趨勢是水面下的洋流。** 大多數分析聚焦在事件(波浪),但決定事件走向的往往是長期趨勢(洋流)。忽略結構趨勢會讓你看到正確的事件但得出錯誤的結論,因為底層條件已經不一樣了。

## 執行步驟
**1. 識別相關的長期趨勢**:哪些結構性力量正在影響當前議題
**2. 評估趨勢方向與速度**:趨勢是加速、減速、還是正在反轉
**3. 評估趨勢對分析的影響**:如何改變行為者的約束、能力與選項
**4. 標記趨勢轉折訊號**:什麼跡象代表趨勢正在改變方向
**5. 整合進事件分析**:將趨勢判斷作為背景約束回饋至相關 SKILL

## 常見結構趨勢類別

| 類別 | 涵蓋範圍 | 典型時間尺度 |
|------|----------|--------------|
| **人口結構** | 老齡化、勞動力變化、城市化、移民 | 10–30 年 |
| **能源轉型** | 化石→再生能源、電動化、核能復興 | 5–20 年 |
| **軍工產能** | 武器生產能力、彈藥庫存重建、新系統部署 | 3–10 年 |
| **聯盟體系** | 聯盟凝聚力、新聯盟形成、不結盟趨勢 | 5–15 年 |
| **技術革命** | AI 軍事化、無人系統、太空、網路 | 3–10 年 |
| **經濟重心** | GDP 比重轉移、供應鏈重組、金融體系演變 | 5–20 年 |
| **制度演變** | 國際秩序轉型、多邊機構功能變化 | 10–30 年 |
| **氣候與資源** | 水資源壓力、糧食安全、極端氣候頻率 | 10–50 年 |

```

```

## 注意事項
- ⚠️ **趨勢不是宿命**——趨勢可以被政策、技術突破或黑天鵝事件打斷
- ⚠️ 注意區分「趨勢」與「時尚」——真正的結構趨勢有底層驅動力,不會因短期事件反轉
- ⚠️ 不同趨勢之間可能互相增強或互相抵消——注意趨勢交互作用
- ✅ 長期趨勢判斷應定期(每月或每季)回顧更新
- ✅ 趨勢轉折訊號應登錄至 SKILL-10 觀測指標
- 💡 最有價值的分析洞察往往來自「事件方向與結構趨勢方向矛盾」的時刻
