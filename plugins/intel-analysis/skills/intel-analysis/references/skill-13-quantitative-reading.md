# SKILL-13|定量數據解讀

## 概述
對量化指標進行趨勢判斷、異常值識別與基準線比對,區分「結構性變化」與「短期噪音」,為定性分析提供數據支撐。

## 對應框架
**擴展模組**|與 SKILL-02(事實建立)、SKILL-10(觀測指標)協同使用

## 核心問題
> 這個數據變化代表什麼?是結構性轉變還是短期噪音?

## 核心原則
> **數據本身不說話,解讀數據的框架才決定意義。** 同一個數字在不同基準線、不同時間尺度、不同背景下可能指向完全不同的結論。必須先建立比較基準,再判斷異常。

## 執行步驟
**1. 識別關鍵指標**:從 SKILL-10 的觀測指標中選取需要量化解讀的數據
**2. 建立基準線**:確定正常範圍(歷史均值、季節性模式、結構性趨勢線)
**3. 判斷偏離程度**:當前值與基準線的偏離幅度、速度、持續時間
**4. 區分訊號與噪音**:用多指標交叉驗證,排除單一指標的誤導
**5. 連結至定性分析**:將數據解讀結論回饋至對應的 SKILL 步驟

## 數據解讀框架

### 三維判斷法

| 維度 | 問題 | 判斷標準 |
|------|------|----------|
| **幅度** | 偏離多少? | 超過歷史 1 個標準差 = 值得注意;2 個 = 顯著異常 |
| **速度** | 多快發生的? | 漸進式(數週)= 可能是趨勢;突發(數天內)= 可能是事件驅動 |
| **持續性** | 持續多久了? | < 1 週 = 可能是噪音;> 2 週 = 可能是結構性 |

### 常用比較基準

| 基準類型 | 用途 | 範例 |
|----------|------|------|
| **歷史均值** | 判斷當前值是否異常 | 布蘭特原油 5 年均價 |
| **季節性模式** | 排除週期性因素 | Q4 能源需求季節性上升 |
| **事件基準** | 與類似歷史事件比對 | 2019 沙特設施遇襲後的油價反應 |
| **結構性趨勢線** | 判斷是否偏離長期方向 | 全球 LNG 產能年增率 |

## 數據解讀模板

```
指標名稱:[]
當前值:[] 日期:[]
基準線:[](類型:歷史均值 / 季節性 / 事件基準 / 趨勢線)

【偏離判斷】
幅度:[偏離 X%,約 N 個標準差]
速度:[漸進 / 突發,時間窗口]
持續性:[已持續 N 天/週]

【交叉驗證】
支持同方向的其他指標:[]
矛盾的指標:[]

【結論】
判斷:[結構性變化 / 事件驅動 / 短期噪音 / 待觀察]
對分析的影響:[回饋至哪個 SKILL 的哪個判斷]
信心水準:[高/中/低] 理由:[]
```

## 輸出格式

```
指標:[] 當前值:[] 基準線:[]
偏離:[幅度/速度/持續性]
判斷:[結構性 / 事件驅動 / 噪音]
交叉驗證:[支持 / 矛盾指標]
回饋至:[SKILL-XX 的哪個判斷]
```

## 注意事項
- ⚠️ **單一指標不足以下結論**,必須至少兩個獨立指標交叉驗證
- ⚠️ 注意「倖存者偏誤」——你看到的數據可能不包含最關鍵的缺失數據
- ✅ 完成後將結論回饋至 SKILL-02(事實建立)或 SKILL-10(觀測指標)
- 💡 與 SKILL-03(可信度評級)配合使用:數據來源也需要評級
