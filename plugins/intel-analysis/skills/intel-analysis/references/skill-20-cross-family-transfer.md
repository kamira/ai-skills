# SKILL-20|跨家族遷移測試

## 概述
強迫對結構不同的危機案件做預測,避免方法論在單一家族內慣性虛高,驗證 SKILL 框架的可遷移性。

## 對應框架
**擴展模組**——每 14 天觸發一次

## 核心問題
> 方法論在陌生案件上還能跑嗎?還是只是在熟悉的故事裡記憶?

## 核心原則
> **低 Brier 不等於高判斷力。** 連續追蹤同一危機家族(同一組行為者+同一地理場景)時,低 Brier 大量來自「對特定行為者的記憶」而非「方法論本身」。記憶不可遷移,方法可遷移——這兩件事在資料上看起來一樣,必須靠刻意設計的壓力測試分開。

## 執行步驟

### 步驟 A:觸發條件
任一成立時必須執行:
- 同一危機家族連續追蹤超過 14 天
- 主家族 Brier 低於 0.15 但「中帶實際發生率 > 75%」
- 連續 3 週方向錯誤率為 0
- 用戶主動觸發(測試方法論是否退化)

### 步驟 B:選題
從以下清單隨機抽取一個非主追蹤家族的近期事件:

| 編號 | 家族 | 範例事件 |
|------|------|----------|
| 1 | **台海/南海** | 解放軍演習、菲律賓灰區事件、台灣選舉 |
| 2 | **印巴** | 喀什米爾衝突、貿易制裁、核子試驗訊號 |
| 3 | **非洲** | 蘇丹內戰、薩赫勒政變、衣索匹亞-厄利垂亞 |
| 4 | **拉美** | 委內瑞拉局勢、墨西哥反毒戰、阿根廷經濟 |
| 5 | **金融市場** | Fed 決策、美元指數、新興市場債務危機 |
| 6 | **科技政策** | 半導體出口管制、AI 監管、加密資產 |
| 7 | **公衛/災害** | 疫情爆發、極端天氣、糧食危機 |
| 8 | **朝鮮半島** | 北韓核試、韓國國內政治、美朝接觸 |

選題要求:
- 不能是當前主追蹤家族的鄰接事件(如追蹤中東就不能挑伊朗-巴基斯坦)
- 必須有可在 14 天內驗證的具體事件節點

### 步驟 C:完整 SKILL 流程跑一次
- 用 SKILL-01〜11 對選定事件做完整慢情報分析
- 至少產出 3–5 筆預測,登錄至預測驗證帳本
- 在帳本加標記「壓力測試題:是」(建議 tags 加 `stress-test`)

### 步驟 D:14 天後對比
計算壓力測試題的 Brier,與主家族同期 Brier 對比:

| 差距 | 判定 | 後續行動 |
|------|------|----------|
| 壓力測試 Brier 高於主家族 ≤ 0.03 | **方法論可遷移** | 維持現有 SKILL 設定 |
| 壓力測試 Brier 高於主家族 0.03–0.07 | **部分慣性虛高** | 主家族總 Brier 扣 0.02 後再評估 |
| 壓力測試 Brier 高於主家族 ≥ 0.08 | **嚴重家族慣性** | 主家族總分視為不可信,必須做結構性檢討 |

### 步驟 E:歸因分析
若壓力測試表現顯著差於主家族,檢查:
- 是否在 SKILL-04 行為者分析時依賴了「主家族行為者卡片」的記憶?
- 是否在 SKILL-05 約束篩選時忽略了陌生家族的特殊約束?
- 是否在 SKILL-07 決策偏好時錯用了「主家族行為者偏好」?

## 跨家族壓力測試模板

```
【測試題目】
日期:
事件:
所屬家族:
與主追蹤家族距離:[相鄰/中等/遙遠]

【完整 SKILL 流程】
SKILL-01〜11 產出(簡化版可接受,但每步必須有)

【預測登錄】
預測 1:[內容] 機率:[%] 驗證窗口:
預測 2:...
預測 3:...
(標記:壓力測試題=是)

【14 天後驗證】
壓力測試 Brier:
主家族同期 Brier:
差距:
判定:[可遷移 / 部分慣性虛高 / 嚴重家族慣性]
歸因:
```

## 輸出格式
- 壓力測試結果應寫入 SKILL-12 週日校準回顧的固定段落
- 連續 2 次「嚴重家族慣性」需觸發 SKILL Manager 結構性檢討

## 注意事項
- ⚠️ 不要選擇你已經追蹤過的鄰接事件——必須真正陌生
- ⚠️ 壓力測試題不能用於對外發表的分析(除非加上明確標記)
- ✅ 完成後將結果歸入 SKILL-12 獨立計算
- 💡 連續 4 次壓力測試表現良好,可考慮把當前主家族擴展為兩個家族並行追蹤
