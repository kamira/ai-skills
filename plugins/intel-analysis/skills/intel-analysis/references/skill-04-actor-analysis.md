# SKILL-04|行為者分析

## 概述
辨識所有關鍵行為者,從利益出發分析其核心目標、利益排序與紅線,與各國立場基準頁交叉比對。

## 對應框架
**步驟三**|十步驟分析框架

## 核心問題
> 誰在局中?每個行為者要什麼?

## 核心原則
> **從利益出發推導行為方向,不從行為反推意圖。**
> 從行為反推意圖容易掉進**鏡像思維**陷阱——你會假設對方跟你有一樣的邏輯和價值觀。

## 執行步驟

**1. 辨識所有關鍵行為者**
- 直接行為者(直接影響事態)
- 間接行為者(受影響但也能反應)
- 隱性行為者(不在檯面但有重大影響)

**2. 針對每個行為者填入行為者卡片**

### 行為者卡片模板

```
行為者:[名稱]

核心利益與戰略目標:
- 在這個問題上追求什麼

利益排序(若多目標衝突,哪個優先):
1.
2.
3.

紅線與底線(絕對不能接受的結果):
-

參照各國立場基準頁:[是否與長期基準一致 / 有何新偏差]
```

**3. 對比各國立場共用基準頁**
- 本次事件是否改變了某個行為者的利益排序?
- 是否出現紅線觸發訊號?

## 常見陷阱

| 陷阱 | 說明 | 避免方法 |
|------|------|----------|
| 鏡像思維 | 假設對方跟自己有一樣的邏輯 | 從對方利益出發,不從自己的合理性判斷 |
| 行為反推意圖 | 看到軍事部署就說「要打仗」 | 先列出所有可能動機(嚇阻、籌碼、演訓等) |
| 漏掉隱性行為者 | 只分析主要國家 | 檢查是否有第三方、非國家行為者、國內派系 |

## 輸出格式

```
【行為者清單】
直接:[]
間接:[]
隱性:[]

【行為者卡片 × N】
(依上方模板逐一填入)
```

## 對手情報體系結構性失靈評估(紀律 15)

若關鍵行為者所依賴的情報體系具備以下任一特徵,該行為者實際看到的世界系統性偏離客觀現實,**進入 SKILL-07 前必須先做「對手認知扭曲修正」**:

- **向上揣摩**:分析師被訓練只報領導想聽的
- **無事後問責**:缺乏解密、調查、公開檢討機制
- **政治忠誠優先**:忠誠於領袖/路線勝過分析正確性

### 評估產出模板

```
對手情報體系特徵勾選:
- [ ] 向上揣摩
- [ ] 無事後問責
- [ ] 政治忠誠優先於分析正確

預期扭曲方向(至少寫一條):
- 高估己方優勢/低估對手意志:[]
- 過早確認結論、缺乏反證輸入:[]
- 哪些結構性反證會被體系過濾掉:[]

對 SKILL-07 的影響:
預測標的=對方【基於扭曲版現實】的最優選擇,而非客觀最優解
```

### 典型案例
- **史達林對巴巴羅薩(1941)**:邊境德軍集結、外交官撤離、間諜直接報告進攻日期等多條訊號被體系過濾——因不符「希特勒不會兩線作戰」的領導判斷
- **俄羅斯 FSB 對 2022 烏克蘭抵抗**:戰前情報被向上修飾為「政府會迅速崩潰、特別軍事行動 3 天結束」
- **CIA 對蘇聯死亡之手系統**:符合鏡像思維(「核控制必在最高層」),外部反證被系統性低估——但這是對手體系失靈的「我方版本」,提醒此紀律雙向適用

> ⚠️ 本紀律與紀律 7(避免鏡像思維)互補但不重疊:鏡像思維問「我有沒有把自己的邏輯套到對方身上」,本紀律問「對方體系本身是否扭曲了他自己的邏輯輸入」。兩者都失守時最危險——你既不理解對方的價值觀,又用「假設對方掌握真實資訊」的前提預測他。

## 注意事項

- 💡 行為者卡片可與各國立場共用基準頁交叉比對,確認本次事件是否偏離長期基準
- 🔍 若行為者屬威權體制 → 必須完成「對手情報體系結構性失靈評估」再進入 SKILL-07
