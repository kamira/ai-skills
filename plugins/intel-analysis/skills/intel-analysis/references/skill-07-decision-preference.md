# SKILL-07|決策偏好與機率調整

## 概述
從所有能打的牌中,透過決策偏好、認知偏誤、組織慣性與國內政治四類因素,收斂出最可能被打出的牌並標記機率。

## 對應框架
**步驟六**|十步驟分析框架

## 核心問題
> 從所有能打的牌(SKILL-05),收斂到最可能被打出的牌。

## 核心原則
> **第二層:決策與認知層**——決策者的認知偏誤、歷史慣性和政治壓力不改變牌桌上有哪些牌,但會**放大或縮小每張牌被選擇的機率**。

## 執行步驟
**1. 彙整可用手段清單**:從 SKILL-05 獲得篩選後的牌桌
**2. 逐項評估四類偏好因素**:對每個手段檢查歷史偏好、認知偏誤、組織慣性、國內政治
**3. 標記機率標籤**:綜合四類因素,為每個手段標記高/中/低/極低
**4. 產出機率調整後清單**:供 SKILL-09 情境預測使用

## 四類偏好調整因素

### 1. 決策偏好與歷史模式
- 這個人或組織過去遇到類似情況怎麼做的?
- 有無明確的決策風格(激進 / 保守 / 拖延)?

### 2. 認知偏誤

| 偏誤 | 說明 | 如何識別 |
|------|------|----------|
| **確認偏誤** | 只看到支持自己判斷的資訊 | 決策者公開言論是否選擇性引用情報? |
| **錨定效應** | 被第一個接收到的資訊鎖定 | 初期錯誤評估是否被長期沿用? |
| **鏡像思維** | 假設對方跟自己一樣理性 | 是否以「換我我不會這樣做」來排除選項? |
| **歷史類比陷阱** | 用過去經驗硬套當前局勢 | 是否用「上次一樣的情況」做類比而忽略差異? |

### 3. 組織慣性
- 官僚體系傾向延續既有做法
- 軍事機構偏好已驗證過的方案
- 檢查是否有制度性因素讓某些選項被系統性偏好

### 4. 國內政治約束
- 民意、選舉時程、派系利益
- 哪些選項在國內政治上代價過高?
- 哪些選項在國內政治上能加分?

## 機率調整模板

```
行為者:[]
可用手段清單:[來自 SKILL-05]

【機率調整】
手段 A:
  歷史偏好:[高/中/低]
  認知偏誤影響:[說明]
  組織慣性:[支持/阻礙]
  國內政治:[支持/阻礙]
  → 綜合機率標籤:[高/中/低/極低]

手段 B:(同上)
```

## 輸出格式

```
行為者:[]
可用手段清單:[來自 SKILL-05]

手段 A → 機率:[高/中/低/極低] 依據:[]
手段 B → 機率:[高/中/低/極低] 依據:[]
手段 C → 機率:[高/中/低/極低] 依據:[]
```

## 注意事項
- ⚠️ 機率調整**不等於排除選項**——「極低」也必須列出
- ⚠️ 判定對方處於**高壓+損失域**時,機率排序須切換為「尾端翻盤潛力最大化」(紀律 12,Prospect Theory)
- ⚠️ 對手屬威權體制時,先做**對手認知扭曲修正**(紀律 15,見 SKILL-04)——預測的是對方基於扭曲版現實的選擇
- ✅ 完成後接 **SKILL-08**(連鎖反應推演)
- 💡 機率標籤會在 SKILL-09 情境預測中正式使用,必須有依據
