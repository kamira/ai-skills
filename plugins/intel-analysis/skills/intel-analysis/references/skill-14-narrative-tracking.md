# SKILL-14|敘事追蹤與輿論分析

## 概述
追蹤各行為者的官方敘事框架變化、媒體議程設定與社群輿論趨勢,將敘事轉向本身視為可分析的訊號與行為者工具。

## 對應框架
**擴展模組**|與 SKILL-04(行為者分析)、SKILL-07(決策偏好)、SKILL-10(觀測指標)協同使用

## 核心問題
> 各方在說什麼故事?敘事框架是否正在轉向?轉向本身代表什麼意圖?

## 核心原則
> **敘事既是訊號也是武器。** 官方敘事的轉向往往先於政策行動——它既反映決策者的認知變化,也是為未來行動建構合法性的工具。追蹤敘事不是「看新聞」,而是分析行為者如何建構現實。

## 執行步驟
**1. 識別各行為者的核心敘事框架**:當前各方用什麼故事解釋局勢
**2. 追蹤敘事轉向**:與前期比較,哪些用語、框架、重點發生了變化
**3. 分析敘事功能**:這個敘事是為了動員、合法化、嚇阻、還是為退場鋪路
**4. 交叉比對行動**:敘事方向與實際行動是否一致(一致 = 增強判斷;矛盾 = 需要解釋)
**5. 評估輿論約束**:國內輿論是否限制或擴大決策者的行動空間

## 敘事分析維度

### 敘事功能分類

| 功能 | 說明 | 範例 |
|------|------|------|
| **動員型** | 為行動建構支持 | 「我們被迫自衛」 |
| **合法化型** | 為已採取的行動辯護 | 「這是符合國際法的必要措施」 |
| **嚇阻型** | 警告對方不要行動 | 「任何挑釁都將遭受毀滅性回應」 |
| **退場型** | 為降級或妥協鋪路 | 「我們已達成主要目標,願意對話」 |
| **轉移型** | 將注意力從某議題移開 | 內部危機時強調外部威脅 |

### 敘事轉向判斷指標

| 指標 | 說明 |
|------|------|
| **用語變化** | 從「敵人」變「對手」、從「絕不」變「在條件下」 |
| **責任歸因移動** | 開始責怪新的對象或淡化原有指控 |
| **時間框架變化** | 從「立即行動」變「長期博弈」 |
| **受眾轉移** | 從國內動員轉向國際溝通 |
| **新敘事元素出現** | 引入之前未使用的論述框架 |

## 敘事追蹤模板

```
行為者:[]
追蹤日期:[]

【當前核心敘事】
主要框架:[用一句話概括]
目標受眾:[國內 / 國際 / 特定行為者]
敘事功能:[動員 / 合法化 / 嚇阻 / 退場 / 轉移]

【敘事轉向追蹤】
前期敘事:[]
變化點:[哪些用語/框架/重點改變了]
轉向方向:[強硬化 / 軟化 / 轉移焦點 / 新框架]
轉向時間:[何時開始觀察到]

【敘事 vs 行動一致性】
敘事方向:[]
實際行動方向:[]
一致性:[一致 / 矛盾]
若矛盾,可能解釋:[]

【輿論約束評估】
國內輿論主流方向:[]
輿論是否限制決策空間:[是/否,說明]
輿論操作跡象:[是否有刻意引導]
```

## 輸出格式

```
行為者:[] 核心敘事:[]
敘事功能:[動員/合法化/嚇阻/退場/轉移]
敘事轉向:[強硬化/軟化/轉移/無變化] 時間:[]
敘事 vs 行動:[一致/矛盾]
輿論約束:[限制/擴大決策空間]
分析影響:[回饋至 SKILL-XX]
```

## 注意事項
- ⚠️ **敘事轉向 ≠ 政策轉向**——可能是試探、內部分歧外溢、或刻意誤導
- ⚠️ 注意區分「官方敘事」與「媒體解讀」與「社群輿論」三個層次
- ✅ 敘事轉向應登錄至 SKILL-10 觀測指標作為獨立訊號
- ✅ 敘事功能分析回饋至 SKILL-04 行為者卡片與 SKILL-07 決策偏好
- 💡 敘事矛盾(說一套做一套)本身就是高價值分析訊號
