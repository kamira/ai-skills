# SKILL-19|機率分布工程

## 概述
專門處理「機率標籤本身的工藝」——確保每個機率數字反映真實判斷而非寫作對沖,避免 resolution 萎縮,維持可區分的機率分布。

## 對應框架
**擴展模組|橫切系統層 E**——強化所有產出機率的步驟(特別是 SKILL-07、SKILL-09),確保「機率標籤」與「真實判斷」一致

## 核心問題
> 這個機率標籤是「真實的不確定」還是「對沖的不敢押」?

## 核心原則
> **Brier Score 衡量校準,但不衡量 resolution。** 一個全部標 0.5 的分析師也能有不錯的 Brier,但完全沒有判斷力。機率分布工程的目標:在維持低 Brier 的同時,**強迫機率標籤分布在三個區間**,避免所有預測都擠在中帶。

四個常見對沖陷阱:
1. **保留下台階對沖**:心裡判斷 70%,寫成 55% 以避免被打臉
2. **讀者觀感對沖**:怕讀者覺得分析太篤定,主動降權
3. **多情境平衡對沖**:人為讓多個情境機率接近 50%,看起來「公平」
4. **語意安全對沖**:用「中(55%)」描述任何不確定的事,把標籤當形容詞而非機率

## 執行步驟

### 步驟 A:機率分解(六維強制分解)

寫總機率前必須先分解六維,每維給 0–1 分數+一行支持證據:

| 維度 | 評估問題 | 分數 |
|------|----------|------|
| **利益強度** | 行為者的核心利益是否強烈支持此行為? | 0–1 |
| **約束剛性** | 五種硬約束是否允許此行為? | 0–1 |
| **時間壓力** | 行為者是否處於必須立即決策的窗口? | 0–1 |
| **損失/獲益域** | Prospect Theory:行為者處於損失域還是獲益域? | 0–1 |
| **內部政治授權** | 國內政治是否支持? | 0–1 |
| **歷史類比基率** | 類似情境下歷史發生率? | 0–1 |

**合成總機率**:算術平均或加權平均(若某維度明顯主導,可給更高權重,但須說明理由)。

### 步驟 B:反 hedging 檢查清單(總機率出來後必過)

- [ ] 我給的機率是否比六維平均值低 10%+?若是,必須寫降權理由
- [ ] 我給的機率是否落在 0.50–0.60?若是,是否真的找到雙向抵消證據?
- [ ] 若被讀者問「為什麼不更篤定」,我能否指出具體反向證據?若不能 → 機率上修
- [ ] 我心裡的真實判斷與寫出的數字是否相同?若不同 → 寫真實判斷

### 步驟 C:分布健檢(情境清單寫完後)

統計本篇所有預測機率,確認分布:

| 區間 | 名稱 | 目標佔比 | 警戒門檻 |
|------|------|----------|----------|
| ≥ 0.65 或 ≤ 0.35 | **敢押區** | ≥ 20% | 低於 20% 視為過度對沖 |
| 0.40–0.60 | **中帶** | ≤ 50% | 高於 50% 視為情境清單過度對沖 |
| ≥ 0.75 或 ≤ 0.20 | **強信號區** | ≥ 1 筆 | 完全沒有視為缺少判斷力展示 |

若分布不達標:
- 中帶過多 → 砍 1–2 個情境合併到主情境,主情境機率上拉到 ≥ 0.65
- 缺強信號區 → 重新檢視是否有真正高機率或極低機率的事件被低估

### 步驟 D:與回測連動(事後校準)

- 每週執行 SKILL-12 後,依各機率帶實際發生率調整下週標籤紀律
- 若「中(55%)」帶連續 2 週實際發生率 ≥ 75%,下週起該帶調整為「中偏高(70%)」
- 若「低(25%)」帶連續 2 週實際發生率 ≤ 5%,下週起該帶調整為「低(15%)」

## 機率分布工程模板

```
【六維分解】
- 利益強度:[0–1]|支持證據:
- 約束剛性:[0–1]|支持證據:
- 時間壓力:[0–1]|支持證據:
- 損失/獲益域:[0–1]|支持證據:
- 內部政治授權:[0–1]|支持證據:
- 歷史類比基率:[0–1]|支持證據:
合成總機率:[%]|合成方式:[平均/加權]

【反 hedging 檢查】
- 與六維平均偏差:[+/-X%]|理由:
- 是否落在中帶(0.50–0.60):[Yes/No]|雙向抵消證據:
- 反向證據能否舉出:[Yes/No]|內容:
- 真實判斷 vs 寫出數字一致性:[Yes/No]

【分布健檢】
本篇預測總數:N
敢押區(≥65% 或 ≤35%)佔比:X% [達標/未達]
中帶(40–60%)佔比:Y% [達標/未達]
強信號區(≥75% 或 ≤20%)筆數:Z [達標/未達]
```

## 輸出格式

在每篇分析的「情境機率」段尾必須有一行:
> 本篇分布健檢:敢押區 X%、中帶 Y%、強信號區 Z 筆

## 注意事項
- ⚠️ 不可用此 SKILL 強行抬高機率——重點是讓寫的數字=真實判斷,不是讓數字變極端
- ⚠️ 若六維分解後判斷確實是「中」,就應該標中——分布健檢只是檢查整篇是否過度集中
- ✅ 與 SKILL-07、SKILL-09、SKILL-12 強制連動使用
- 💡 連續 4 週通過分布健檢後,可降低檢查頻率為每兩週一次
