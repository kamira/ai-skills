# SKILL-15|制度與法律約束分析

## 概述
分析國際法、條約義務、國內法律程序與制度性約束如何限制或賦能行為者的行動空間,作為 SKILL-05 硬約束篩選的專業化延伸。

## 對應框架
**擴展模組**|與 SKILL-05(可用手段篩選)的政治授權約束深度整合

## 核心問題
> 法律和制度框架允許什麼、禁止什麼?繞過這些約束的成本是多少?

## 核心原則
> **制度約束不是絕對的牆,而是有成本的門檻。** 行為者可以選擇違反制度約束,但必須承擔相應的政治、法律與聲譽成本。分析的重點不是「能不能做」,而是「做了要付出什麼代價」。

## 執行步驟
**1. 識別相關法律與制度框架**:國際法、雙邊條約、國內法、組織章程
**2. 評估約束強度**:硬約束(違反代價極高)vs 軟約束(可繞過或重新解釋)
**3. 評估繞過成本**:違反或繞過此約束需要付出什麼政治、法律、聲譽代價
**4. 檢查先例**:歷史上是否有行為者成功繞過類似約束
**5. 回饋至 SKILL-05**:將結論整合進可用手段篩選的約束檢查

## 制度約束分類

| 約束類型 | 說明 | 範例 |
|----------|------|------|
| **國際法** | 國際習慣法、日內瓦公約、海洋法 | 對平民目標的攻擊限制 |
| **條約義務** | 雙邊或多邊條約承諾 | NPT 不擴散條約、北約第五條 |
| **安理會決議** | 具約束力的 UNSC 決議 | 制裁決議、武力授權決議 |
| **國內法律程序** | 國內立法機構的授權要求 | 美國 War Powers Act、國會宣戰權 |
| **組織章程** | 國際組織的內部規則與程序 | EU 一致決、NATO 共識決 |
| **非正式制度** | 不成文但被遵守的慣例與規範 | 核武禁忌、化武紅線 |

### 約束強度評估

| 強度 | 說明 | 違反代價 |
|------|------|----------|
| **硬約束** | 違反代價極高,幾乎無法承受 | 國際孤立、制裁、軍事干預 |
| **中度約束** | 違反代價顯著但可承受 | 聲譽損害、外交抗議、部分制裁 |
| **軟約束** | 可透過重新解釋或程序變通繞過 | 法律爭議、政治批評 |
| **名義約束** | 形式上存在但實際執行力極低 | 口頭譴責 |

## 制度約束分析模板

```
行為者:[]
擬議行動:[]

【相關法律與制度框架】
框架 1:[名稱] 類型:[國際法/條約/國內法/...]
  約束內容:[具體限制什麼]
  約束強度:[硬/中/軟/名義]
  違反代價:[具體說明]

框架 2:(同上格式)

【繞過可能性】
是否有合法繞過路徑:[是/否] 說明:[]
歷史先例:[是否有成功繞過的先例]
繞過成本:[政治/法律/聲譽代價]

【結論】
此行動在制度面:[完全合法 / 灰色地帶 / 需要特殊授權 / 明確違反]
約束對行動可行性的影響:[不影響 / 提高門檻 / 需要額外步驟 / 實質阻擋]
回饋至 SKILL-05:[更新哪個約束判斷]
```

## 輸出格式

```
行為者:[] 擬議行動:[]
相關框架:[名稱] 約束強度:[硬/中/軟/名義]
繞過可能性:[有/無] 繞過成本:[]
結論:[合法/灰色/需授權/明確違反]
回饋至 SKILL-05:[更新內容]
```

## 注意事項
- ⚠️ **法律解釋不是唯一的**——不同行為者對同一條法律可能有完全不同的解釋
- ⚠️ 不要假設「違法 = 不會做」,要評估的是違反的成本而非可能性
- ✅ 結論回饋至 SKILL-05 的政治授權約束
- 💡 制度約束也可能被行為者主動利用——用法律程序拖延時間或建構合法性
